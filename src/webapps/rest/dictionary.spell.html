dictionary.spell
REST API . dictionary.spell

Runs a fuzzy search to find the simillar spelling words. For example a word of bizasys which is a spelling mistake (a instead of o) will get rectified using this API.
One can use this in case of no results found hoping it is a fat fingering at user end.

Parameters


Name Type Description
word String A given word.
hkey String The API Key. If you don't have an account key, You can Create A Key Here.

Response


On sucess it returns word listing. The XML format is as below

<result>
    <string>without</string>
</result>

In case of error, the error message is inside XML. An error message format is,
The Error response format is, 

        <?xml version="1.0" encoding="UTF-8"?>
        <result>
            <error>
                <msg>Failure : SearchService: ...
            </error>
        </result>
